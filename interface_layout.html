Interface/ Layout



“As far as the customer is concerned, the interface is the product.” – Jef Raskin, Interface expert at Apple Inc (circa 1970).

Projects in this realm of design are the best as it is a nice balance between working in the idea/ solutions phase of the project (with client/ UX) and working directly with tech to make sure that the design supports the function. I see myself as a digital designer as well as a problem solver/ creative thinker and therefore interface design really makes me happy.

I have put layout and interface design in the same category as I think both revolve around a sense of space architecture and flow, although interface design is of course meant to be more functional.

This was a team project
